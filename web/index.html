Terminal
$ git clone https://github.com/BluShooz/insurance-revenue-engine.git

$ npm install
$ npm run db:setup
$ npm run dev dashboard:overview

Powerful Features

Everything you need to manage your insurance business in one place

Lead Management

Track prospects from initial contact to closed sale with full lifecycle management.

Dynamic Lead Scoring

AI-inspired algorithm scores leads 0-100 based on intent, status, activities, and outcomes.

Activity Logging

Record all interactions with 15+ activity types. Automatic status updates based on activities.

Policy Management

Full lifecycle tracking from quote to issued. Support for all product types.

Commission Tracking

Automatic commission calculation on policy issue. Track pending and paid commissions.

Automation Triggers

Built-in workflows for lead creation, activity logging, policy issuance, and more.

CLI Commands

Simple, intuitive commands for every action

Lead Management

npm run dev lead:add "John" "Doe" "555-1234"
npm run dev lead:list
npm run dev lead:hot

Activity Logging

npm run dev activity:log <leadId> CALL_INBOUND
npm run dev activity:list <leadId>
npm run dev activity:stats

Policy Management

npm run dev policy:add <leadId> "Carrier" TERM_LIFE
npm run dev policy:list
npm run dev policy:update-status <id> ISSUED

Commission Tracking

npm run dev commission:summary
npm run dev commission:view
npm run dev commission:pay <policyId>
